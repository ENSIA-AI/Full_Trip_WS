1. Project Description
Our website is a Travel Booking Platform that allows users to book flights, hotels, car rentals,
attractions, and full travel packages published by travel agencies.
2. Goal of the Website
- Provide a centralized platform for trip planning.
- Allow travel agencies to publish and manage packages.
- Help users compare travel offers.
- Offer smooth booking across multiple services.
3. Target Audience
Individuals:
- Age: 18–55
- Gender: Both
- Country: Algeria and international
- Areas: Urban/Semi-urban
- Income: Middle income
- Education: High school+/university
- Occupation: Students, employees, freelancers
- Device: Mainly smartphones and laptops
Companies (Travel Agencies):
- Size: Small/medium agencies
- Visitors: Managers, travel agents

- Purpose: Manage and promote packages
4. Why People Visit the Website
- Book flights, hotels, cars, attractions, or packages
- Discover offers and compare prices
- Access time-sensitive information
- Contact agencies
